# Go4Search

Golang based search engine implementation.

## Implemented Features

* Indexing
    * Inverted Index
* Search
    * TF-IDF
    * BM25
* Natural Language Processing
    * Subword Tokenization
    * Stopword Removal
    * Language Detection
